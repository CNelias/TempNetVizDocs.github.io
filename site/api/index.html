Temporal Graph Visualization – API Reference

This module provides utilities for visualizing, analyzing, and animating temporal graphs.
You can explore your data interactively through the GUI (python -m tempgraphviz.main_gui) or directly use the functions below.

display_graph(path_to_file, **kwargs)

Visualizes a single temporal graph layer or an aggregated view of the network.

Parameters

Name Type Description
path_to_file str Path to the input file containing the temporal graph data.
percentage_threshold float, optional Edge-weight threshold below which edges are removed. Default 0.0.
mnn int or None, optional Number of mutual nearest neighbors to include.
mutual bool, optional If True, keeps only mutual connections.
avg_graph bool, optional If True, displays the time-averaged graph.
affinity bool, optional If True, uses affinity weights instead of distances.
node_metric str, optional Node sizing metric (e.g., "degree", "strength").
node_cmap matplotlib colormap, optional Node color map.
scale_edge_width bool, optional Whether to scale edge thickness by edge weight.

Returns

matplotlib.axes.Axes — Axis with the plotted graph.

Example

from tempgraphviz import display_graph

display_graph("data/temporal_graph.npy", node_metric="degree", node_cmap="viridis")

display_stats(path_to_file, **kwargs)

Plots key statistics (e.g., degree, density, clustering) across time for the temporal graph.

Parameters

Name Type Description
path_to_file str Path to temporal graph data file.
percentage_threshold float, optional Edge threshold for filtering.
metrics list[str], optional List of metrics to compute (e.g., ["degree", "density"]).
show_plot bool, optional If True, displays the plot immediately.

Returns

dict — Computed statistics for each metric over time.

Example

from tempgraphviz import display_stats

stats = display_stats("data/temporal_graph.npy", metrics=["density", "clustering"])

display_animation(path_to_file, **kwargs)

Generates an animation showing how the graph evolves over time.

Parameters

Name Type Description
path_to_file str Path to temporal graph file.
fps int, optional Frames per second for the animation.
save_path str, optional If provided, saves the animation to the given path (e.g., .mp4).
node_cmap matplotlib colormap, optional Node color scheme.
scale_edge_width bool, optional Scale edge thickness by edge weight.

Returns

matplotlib.animation.FuncAnimation — Animation object.

Example

from tempgraphviz import display_animation

display_animation("data/temporal_graph.npy", fps=10, save_path="animation.mp4")

plot_temporal_layout(path_to_file, ax=None, percentage_threshold=0.0, mnn=None, avg_graph=False, affinity=True, rm_fb_loops=True, mutual=True, rm_index=True, **kwargs)

Displays the temporal layout of a graph, aligning nodes vertically across time and plotting edges between them.
Each column represents a time step, and each row corresponds to a node.

Parameters

Name Type Description
path_to_file str Path to temporal graph file.
ax matplotlib.axes.Axes, optional Axis to draw on. If None, a new figure is created.
percentage_threshold float, optional Edge-weight threshold for filtering.
mnn int or None, optional Mutual nearest neighbors parameter.
avg_graph bool, optional Whether to use the average graph for node ordering.
affinity bool, optional If True, treats data as affinities (not distances).
rm_fb_loops bool, optional Remove feedback (self) loops.
mutual bool, optional Keep only mutual connections.
rm_index bool, optional Whether to remove node index labels.
node_labels bool, optional Display node labels.
node_size float, optional Default node size.
node_metric str, optional Node size metric.
node_cmap matplotlib colormap or "none", optional Node color map.
edge_width float, optional Default edge width.
edge_cmap matplotlib colormap, optional Edge color map.
scale_edge_width bool, optional Scale edge width by edge weight.

Returns

matplotlib.axes.Axes — Axis containing the plotted temporal layout.

Example

from tempgraphviz import plot_temporal_layout

plot_temporal_layout("data/temporal_graph.npy", node_labels=True, node_metric="degree")

Notes

  • All visualization functions accept **kwargs for fine-tuning visual properties.
  • The module supports .npy, .csv, or .txt temporal adjacency matrices.
  • For interactive use, run: bash python -m tempgraphviz.main_gui
